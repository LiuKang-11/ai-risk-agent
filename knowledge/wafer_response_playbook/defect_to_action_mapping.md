# Defect to Action Mapping

## Scratch
- prioritize handling-path review
- request tool or robot inspection
- notify equipment engineering

## Edge-Ring or Edge-Loc
- review edge hardware, edge process settings, and perimeter metrology
- involve process engineering and equipment engineering when hardware evidence exists

## Donut or Center
- request chamber, recipe, and radial metrology review
- compare recent chamber history and similar cases

## Random
- prioritize contamination assessment
- compare recent lots and shared tool paths
- consider metrology follow-up and containment review

## Near-full
- treat as urgent containment candidate
- notify broad owner set
- submit lot-hold request when policy allows

## Loc
- request localized process or hardware review
- compare hotspot location recurrence

## None
- log only if needed for trend tracking
- do not overreact without corroborating evidence
