# Mixed Pattern Guidance

## When to mark a case as mixed
- top two classes are both plausible
- extracted features support more than one geometry strongly
- wafer map shows separate regions with different pattern types

## Mixed-pattern workflow
- identify dominant class
- record secondary class
- explain which evidence supports each
- rank causes that could explain both patterns or a sequence of events

## Common mixed cases
- `Scratch` plus `Random`: mechanical damage with contamination side effects
- `Edge-Ring` plus `Random`: edge process issue with contamination
- `Center` plus `Donut`: radial non-uniformity transitioning over time
- `Loc` plus `Random`: isolated hotspot inside a generally unstable process

## Escalation hint
Mixed patterns should increase uncertainty and often justify metrology follow-up or human engineering review.
